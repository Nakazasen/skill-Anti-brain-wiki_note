## Metrics

Measure the same fields for all three workspaces:

- Install success: wheel reinstall exits `0`.
- Version correct: `abw --workspace <PATH> version` reports package version `0.3.13`.
- Doctor health: `doctor` completes and reports actionable status without crashing.
- Ingest success: `ingest raw` completes or fails with a clear bounded error; no silent query fallback.
- Duplicate drift: count duplicate/sibling draft creation after repeated ingest.
- Conflict noise: count new conflict reports and distinguish new reports from historical conflict files.
- Retrieval grounding: targeted query should cite local wiki/source evidence or log a knowledge gap.
- Bounded summary behavior: broad summary should state bounded/partial scope or insufficient evidence.
- Speed: record wall-clock time for install, doctor, ingest, dashboard, targeted query, and broad summary.
- Upgrade compatibility from old ABW state: record whether old `.brain`, `processed`, `drafts`, and `wiki` are read safely and preserved.

## Pass / Fail Thresholds

Pass thresholds:

- Install exits `0` and the active command resolves to package version `0.3.13`.
- `version` and `doctor` complete for all three workspaces.
- No command modifies engine code, `VERSION`, `pyproject.toml`, or tests.
- Existing `.brain`, `processed`, `drafts`, `wiki`, and `raw` data remain present after validation.
- `ingest raw` either completes or returns a precise, non-destructive error for empty/unsupported raw state.
- Repeated ingest does not create duplicate canonical drafts for the same unchanged source.
- New conflicts are reported separately from historical conflict files.
- Targeted retrieval is grounded in local evidence or logs an explicit gap.
- Broad summary answers are bounded partial summaries or explicit insufficient-evidence responses.

Fail conditions:

- Wheel reinstall fails or version is not `0.3.13`.
- `doctor`, `version`, `dashboard`, targeted retrieval, or broad summary crashes.
- `ingest raw` silently routes to unrelated query behavior.
- Existing workspace state is deleted or rewritten outside expected ABW runtime outputs.
- Broad summary claims full-corpus reasoning without bounded evidence.
- Old ABW state blocks validation without a clear diagnostic.
